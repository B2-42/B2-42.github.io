# Visor de obras (GitHub Pages)

He añadido una página `obra.html` que muestra información de una obra cargando un JSON desde `/obras/<id>.json`.

Pasos para añadir obras:
1. Crear un archivo JSON en la carpeta `obras/` con un id único, por ejemplo `mi-obra.json`.
2. El QR que crees debe apuntar a: `https://b2-42.github.io/obra.html?id=mi-obra`

Control de accesos:
- Solo las personas con permisos de escritura en este repositorio podrán modificar/añadir archivos. Si este repositorio es tuyo y no has añadido colaboradores, entonces solo tú podrás modificarlo.
- Si quieres añadir contribuciones de otras personas pero revisarlas antes de publicar, puedo añadir instrucciones para usar Pull Requests y protección de rama.

Si quieres que suba más obras, cambie colores o que cada obra tenga una imagen de fondo, dímelo y lo actualizo.
